# 🧠 Estudo sobre Mentalidade Financeira

## 📌 Contexto
Este projeto tem como objetivo investigar os padrões de comportamento e mentalidade financeira que influenciam a construção de riqueza ou a permanência em ciclos financeiros limitantes.

A análise foi realizada com apoio de Inteligência Artificial (NotebookLM), utilizando fontes reais como livros, vídeos e conteúdos especializados, com foco em transformar informação em conhecimento aplicável.

## 🎯 Objetivos
- Identificar padrões mentais que impactam a vida financeira
- Analisar diferenças entre comportamentos de pessoas ricas e pobres
- Extrair insights estratégicos a partir de múltiplas fontes
- Utilizar IA como ferramenta ativa de aprendizagem e síntese
